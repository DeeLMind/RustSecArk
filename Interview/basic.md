# 基础知识

## 所有权（Ownership）和借用（Borrowing）
请解释 Rust 中的所有权、借用以及可变借用的概念，并说明它们是如何保证内存安全的。你可以举一个简单的代码示例来说明。

### 1. 所有权（Ownership）
- 每个值在 Rust 中都有一个唯一的**拥有者**（owner）。
- 当拥有者**超出作用域**时，值会自动被释放（调用 `drop`），不需要手动释放内存。
- 值的**所有权可以转移**（move），这时原来的变量不再有效。

### 2. 借用（Borrowing）
- 借用允许其他变量临时访问某个值而不获取所有权。
- 借用分为：
  - **不可变借用**（`&T`）：可以同时存在多个不可变借用，但不能修改值。
  - **可变借用**（`&mut T`）：一次只能有一个可变借用，允许修改值，但同时不能有不可变借用。
- Rust 的借用规则通过编译期检查保证：
  - 数据不会在使用时被释放。
  - 不会有数据竞争或悬垂指针。

## 示例代码

```rust
fn main() {
    let mut s = String::from("Hello"); // s 是拥有者

    // 不可变借用
    let r1 = &s;
    let r2 = &s;
    println!("{} and {}", r1, r2);

    // 可变借用
    let r3 = &mut s;
    r3.push_str(", world!");
    println!("{}", r3);

    // 超出作用域，所有权自动释放
}
